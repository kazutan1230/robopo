## 採点 URL について

採点者が採点するページ URL には、大会[competitionID]、採点者[umpireID]、コース[courseID]、選手[playerId]が含まれている。
現時点で採点者[umpireID]とコース[courseID]が 1 対 1 で対応せさているので大会[competitionID]、採点者[umpireID]、選手[playerId]であり、URL

`[RootURL]/challenge/[competitionId]/[UmpireId]/`

で選手一覧を表示し、

`[RootURL]/challenge/[competitionId]/[UmpireId]/[playerId]`

で採点するようにしている。
